# EmbATlink
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一款标准化、高通用性的AT指令驱动框架，解决物联网项目多模块AT指令交互的代码冗余与维护难题。

## 版本说明
- v1.0_20260227：完成全部核心功能，支持多设备并行、指令重发、被动监控、格式化发送等能力

## 项目介绍
EmbATlink采用分层解耦设计，抽象底层硬件接口，适配任意MCU平台，可快速集成到4G/WiFi/蓝牙等物联网模组的AT指令交互场景。
